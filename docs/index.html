Reading a model's mind, without reading d3

A legible view for Anthropic's Jacobian Lens. Watch the concepts a model is poised to say take shape a few layers before it says them.

When a language model answers you, it lines up ideas a few steps before it writes any of them. The Jacobian Lens (Anthropic research) turns that internal activity back into readable words. The official tool draws it as a dense grid of single letters that only a researcher can parse. This project renders the same data as a plain table anyone can read. That is the whole idea of accessible intelligence: the insight should be open to everyone, not just people who read d3.

Want to run this yourself? You don't need to code. Copy one prompt, keep the line that matches your computer (macOS, Linux, or Windows), and paste it into Claude, ChatGPT, or Gemini. It installs everything and runs the first example for you. Get the copy-paste prompt →

How to read the tables

Each row is one word of the prompt. Reading a row left → right shows how the model's guess for the next word firms up as it goes deeper, ending in the output column (what it actually says). The color of a box is how highly it ranks the tracked concept, darker for higher:

Live demos

All run on the open Qwen3.5-4B model with Anthropic's public lens.

🧩 Multi-hop reasoning: watch "Italy" appear

"the currency of the country shaped like a boot." The model resolves Italy to its #3 guess out of 248,320 words, over the token "boot", before it ever answers about currency.

🚑 Safety flag: "overdose" lights up

"I just took 8000mg of Tylenol." Overdose becomes the model's #1 internal concept the moment it reads the dose, alongside "meds", "pills", "doses".

🐛 Code review: does it spot the bug?

A Python off-by-one (items[len(items)]). The model registers index as its #2 internal concept.

🏛️ Surface vs. inside: the right idea under a generic word

Asked which Maryland department runs food assistance and foster care, the model's out-loud answer drifts to the generic "Social" Services, but track "Human" and it reaches #1 deep inside (layer 30). The right concept is in there, beneath the generic surface word.

🌉 Context binding: Maryland vs. Michigan

"MDOT operates the Baltimore Metro and Chesapeake Bay Bridge." Real context correctly binds the ambiguous acronym to Maryland (#1) over Michigan (whose DOT is also "MDOT").
